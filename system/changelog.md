# PromptPolish Changelog

All notable changes to this project will be documented in this file.

---

## [v1.0.0] - Initial Release
**Date:** 2025-11-17

### Added
- Created main `system_prompt.md` — core behavior.
- Added `error_handling.md` — fallback and safety rules.
- Added `output_template.md` — standardized response pattern.
- Added `metadata.md` — high-level project description.
- Added `examples.md` — sample inputs and outputs.
- Created initial `prompts/` directory with examples.
- Added minimal helper file structure under `src/`.

### Purpose
This is the first complete version of **PromptPolish**, a minimal and user-friendly prompt rewriting engine built for:
- English-only output  
- Tone adjustment  
- Clarity improvement  
- Beginner-friendly use  
- Low-code / no-code environments
