# PromptPolish Metadata

This file contains the high-level metadata about the PromptPolish system — helpful for documentation, versioning, and clarity.

---

## Project Name
**PromptPolish**

## Version
**v1.0.0**

## Description
PromptPolish is a lightweight English-only prompt-rewriting engine designed to adjust tone, clarity, and structure of user-provided text without requiring coding or complex configurations.

---

## Supported Tones
- Friendly  
- Interview-appropriate  
- Empathetic  
- Corporate  
- Assertive  

All tones are designed to maintain meaning while shifting personality and expression.

---

## Core Principles
1. **Warm + Uplifting Personality**  
2. **Light Professionalism**  
3. **No Over-complication** — simple rewrites  
4. **English-only focus**  
5. **No hallucination** — never add facts  
6. **Respect user boundaries + safety**

---

## Primary Components
- `system_prompt.md` → main system instructions  
- `error_handling.md` → fallback behavior  
- `output_template.md` → formatting rules  
- `tones.json` → tone definitions  
- `examples.md` → working examples  
- `prompts/` → user-ready prompts  
- `src/` → minimal helper logic (optional)

---

## Intended Audience
- Beginner prompt engineers  
- Content creators  
- Students preparing for interviews  
- Anyone needing polished English outputs
